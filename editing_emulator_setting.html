Android Emulator की Settings को Edit करने का तरीका

अगर आपका Android Emulator धीमा चल रहा है, या आप उसका RAM, CPU, Screen Size, Graphics, या Storage बदलना चाहते हैं, तो आपको Emulator Settings को edit करना होगा। नीचे दिए गए steps को follow करें:

Step 1: AVD Manager खोलें

  • Android Studio खोलें।
  • Tools → AVD Manager पर जाएं।
  • आपके बनाए हुए Virtual Device (Emulator) की list दिखाई देगी।

Step 2: Emulator की Settings Edit करें

  • जिस Emulator को एडिट करना है, उसके आगे Pencil Icon (✏ Edit Button) पर क्लिक करें।
  • अब आपको Hardware और Performance Settings दिखेंगी।

Step 3: RAM और CPU को Optimize करें

RAM बढ़ाएं

  • Default RAM 1536MB (1.5GB) होती है, इसे 2048MB (2GB) या 4096MB (4GB) कर सकते हैं।
  • Best Performance: अगर आपके पास ज्यादा RAM है, तो इसे बढ़ा दें।

CPU Cores बढ़ाएं

  • Performance के लिए CPU को 2 या 4 Cores पर set करें।
  • अगर आपका लैपटॉप या PC ज्यादा Cores सपोर्ट करता है, तो आप इसे बढ़ा सकते हैं।

Step 4: Graphics और Rendering को Optimize करें

Graphics Mode:

  • Hardware - GLES 2.0/3.0 (अगर आपका system support करता है तो इसे चुनें)
  • Software Rendering (अगर Hardware Rendering से Emulator crash हो रहा हो)

Device Frame को Disable करें

  • इससे Emulator थोड़ा Fast चल सकता है।

Use Host GPU

  • इस ऑप्शन को Enable करने से ग्राफिक्स तेज़ी से Render होंगे।

Step 5: Internal Storage और SD Card Space बढ़ाएं

Internal Storage बढ़ाएं:

  • Default में 800MB से 2GB तक मिलता है, इसे 4GB या 8GB कर सकते हैं।

SD Card Size सेट करें:

  • अगर आपको Emulator में ज्यादा Data Save करना है, तो SD Card को 2GB+ पर सेट करें।

Step 6: Screen Resolution और DPI को Edit करें

Resolution बदलें:

  • Default में 1080x1920 (xxhdpi) रहता है।
  • अगर आपका सिस्टम Slow है, तो इसे 720x1280 (hdpi) कर दें।

DPI कम करें:

  • Default 420dpi रहता है, इसे 320dpi या 280dpi कर सकते हैं।
  • Low DPI से Performance बेहतर हो सकता है।

Step 7: Boot Mode को Optimize करें

Quick Boot को Enable करें:

  • Cold Boot धीमा होता है, इसे Quick Boot में बदलें।
  • इससे Emulator Fast Load होगा।

Step 8: Save और Restart करें

  • Apply पर क्लिक करें।
  • Finish दबाएं।
  • Emulator को Restart करें और Check करें कि Performance Improve हुआ या नहीं।

Step 9: Command Line से Emulator Settings बदलें (Optional)

  • अगर आप Command Line से Emulator Settings बदलना चाहते हैं, तो:
  • SDK Path पर जाएं:
cd ~/Android/Sdk/emulator

Emulator की RAM और Storage सेट करें:

emulator -avd Pixel_6_API_34 -memory 4096 -partition-size 8192

(Pixel_6_API_34 को अपने Emulator के नाम से बदलें)

Android Emulator के Useful Shortcuts

अगर आप Android Emulator को तेज़ी से इस्तेमाल करना चाहते हैं, तो ये Keyboard Shortcuts आपकी Productivity बढ़ा सकते हैं!

Basic Emulator Shortcuts

Emulator Start/Stop Ctrl + E Cmd + E
Power Button (Screen On/Off) Power Key Power Key
Volume Up Ctrl + ↑ Cmd + ↑
Volume Down Ctrl + ↓ Cmd + ↓
Rotate Left (Counterclockwise) Ctrl + ← Cmd + ←
Rotate Right (Clockwise) Ctrl + → Cmd + →
Zoom In Ctrl + = Cmd + =
Zoom Out Ctrl + - Cmd + -
Toggle Fullscreen Mode F11 Cmd + F
Take Screenshot Ctrl + S Smd + S
